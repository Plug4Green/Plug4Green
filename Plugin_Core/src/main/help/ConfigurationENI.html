How do I configure the plug-in for use at ENI?

You have to configure the FIT4Green properties file f4gconfig.properties as follows:

  • Section: COM NAMES MAPPING
    1. comNames=comENI
    2. comEni=org.f4g.com.ComEni

You have to configure the ComEni properties file ComEni.properties as follows:

  • Section: ENI Configuration, configure the data to access the VMWare Web Services of the Data Centres
    1. datacenter=number_of_Data_Centres ; the number of Data Centres with different VMWare Web Services configurations
    2. datacenter_0=name_of_the_Data_Centre ; the name of the Data Centre
    3. url_name_of_Data_Centre=URL_of_Web_Services ; the URL for the Web Services.
    4. username_name_of_Data_Centre=user_name_encrypted
    5. password_name_of_Data_Centre=password_encrypted ; the user name and password used to access the VMWare Web Services
    6. isSimulated=boolean ; the flag used to set the ENI communication module in simulation mode (not implemented)
    7. interval=milliseconds ; the interval for taking values from VMWare
    8. port=port_number ; the port used for the SSH server inside ENI communication module
    9. keystore= path_to_file ; the file with the keys to secure accessing the Web Services
    10. keystorePass==keystore_password_encrypted ; the password for the keystore file
    11. RetryCount=number_of_retries ; the number of retries when losing connection with the Web Services before communicate the lost of connection to FIT4Green plug-in
  • Section: Servers configuration, configure the data to access to the ILOs of each server on the Data Centre
    1. hostPort=port_number ; the port for SSH to connect with the servers ILOs
    2. server_name_ILO=ILO_IP ; the IP of the server's ILO
    3. server_name_DC=Data_Cenre_name ; the name of the Data Centre where the server is located.
    4. user_server_name_ILO=user_encrypted ; the user name to connect to the server's ILO
    5. pass_server_name_ILO=password_encrypted ; the password to connect to the server's ILO

How do I encrypt the user names and passwords?

You can encrypt the user names and passwords by using the tool EncryptString. Run this tool with the command: 

java EncryptString word_to_encrypt

The result printed on the screen is the word encrypted.

Remember: Use only alphanumeric characters within the words that you want to encrypt.

This tool is located at the root of the FIT4Green code repository.
